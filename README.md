# Portfolio Website

This repository hosts the code for my personal portfolio website, designed to effectively showcase my skills, projects, and professional journey. The site serves as a digital business card and a product in itself, aimed at delivering a smooth, engaging user experience for recruiters, collaborators, and clients.

## Product Vision

To provide a clean, intuitive, and responsive platform that highlights my expertise, experience, and key achievements while making it easy for visitors to connect and explore my work. The portfolio is built with scalability in mind, allowing for future feature additions and enhancements.

## Key Features & User Benefits

- **Clear Professional Narrative:** Organized sections such as About, Skills, and Projects provide a logical flow for users to understand my background and capabilities.
- **Responsive Design:** Ensures a seamless experience across devices, maximizing accessibility for all users.
- **Interactive Chatbot:** Engages visitors in real-time, enabling quick questions and improved user interaction.
- **SEO Optimized:** Includes robots.txt and sitemap.xml to enhance discoverability by search engines.
- **Contact & Networking:** Easy access to my professional links and contact information facilitates connections and opportunities.

## Technology Stack

- Core frontend technologies: HTML5, CSS3, JavaScript  
- Static assets optimized for fast loading  
- No backend dependencies for simplicity and reliability

## Project Structure

The project structure is organized as follows:

portfolio-website/
│
├── index.html # Main entry point of the website
├── styles.css # Global styles and responsive layout
├── Assets/ # Images, icons, and other static assets
├── Skills/ # Skill-related content and visuals
├── chatbot/ # Chatbot UI components and logic
├── my_profile.md # Profile content in markdown format
├── robots.txt # Instructions for search engine crawlers
├── sitemap.xml # Sitemap to help search engines index the site
└── README.md # Project documentation


## How to Use

Clone the repo locally and open `index.html` in any modern browser to view the portfolio. The site requires no build process or backend server, allowing for quick iterations and easy deployment.

```bash
git clone https://github.com/rishavju21/portfolio-website.git
cd portfolio-website
open index.html

Deployment & Scalability

The site can be hosted on any static site platform like GitHub Pages, Netlify, or Vercel, supporting rapid deployment and continuous updates. The modular structure supports adding new features like blog integration, analytics, or API-driven content.
